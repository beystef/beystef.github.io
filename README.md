# Personal Website Template

This repository now contains a simple static personal website template that works well with GitHub Pages.

## Files

- `index.html`: homepage with biography, projects, writing, and contact sections
- `blog/index.html`: simple blog archive page
- `styles.css`: shared site styles
- `script.js`: renders homepage content from shared data
- `data/site-data.js`: the main file you can edit later with your own information

## Customize

Most personal content lives in `data/site-data.js`.

You can update:

- your name, role, short bio, and contact links
- featured projects
- blog posts
- timeline items
- skills/interests

## GitHub Pages

If this repository is published with GitHub Pages, the site will serve directly as a static website with no build step required.
